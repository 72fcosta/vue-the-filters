# Vue The Filters
Filters for Vue

### :bookmark: Date Pretty
Prettify Date

#### Usage
```js
datePretty
```

### :bookmark: Datetime Pretty
Prettify Datetime

#### Usage
```js
datetimePretty
```

### :bookmark: Upper First
First letter capitalized in all words

#### Usage
```js
upperFirst
```

## Installation

#### Install via NPM
```sh
$ npm install vue-the-filters --save
```

#### Install via Yarn
```sh
$ yarn add vue-the-filters
```
